# リッツプロ (Ritz Pro) ナレッジ概要

## 1. 事業定義
株式会社Fプロダクションにおける、**FC（フランチャイズ）開発支援および経営コンサルティング事業**の名称。

### 事業の2本柱
1. **本部構築**: フランチャイズ本部の仕組み作り、マニュアル化、契約整備。
2. **加盟店開発**: 加盟希望者の募集、選定、契約獲得。

古橋氏は、上記両業務において**カスタマーサクセス（CS）への指示出し・統括**を担う。

## 2. チーム体制・主要パートナー
- **統括**: 古橋寛
- **資料作成・パートナー**: **久保田氏**（主に資料作成、ドキュメント整備を担当）
- **協力パートナー**: 小笠原氏（美容サロン領域）
- **現在の状況**: 美容サロン支援において、以前のメイン担当者の交代に伴い、現在は古橋氏が主導して対応を行っている案件がある。

## 4. 戦略的位置づけ
- **Fプロダクションの柱**: Fプロダクションにおける高単価案件の多くは「リッツ」経由で獲得されている。
- **強み**: 
    - リッツのネットワークを活用した高額案件へのアクセス。
    - FC構築のノウハウ。
- **課題**: 
    - リッツ経由の案件獲得をいかに安定化・仕組み化できるか。
    - パートナーシップの最適化。
