# 久保田氏ミーティング準備メモ（2026/01/07）

## ミーティングの目的
FC開発支援の2大業務「本部構築」と「加盟店開発」において、古橋氏がカスタマーサクセス（CS）へ効率的に指示出しを行えるよう、久保田氏に資料整備・仕組み化を依頼する。

---

## 1. 【本部構築】に関する依頼事項
本部構築は「仕組みの言語化」が肝です。久保田氏には以下のドキュメント化を依頼します。

- **「本部構築ロードマップ」の標準化**
    - 各クライアントの進捗を一目で把握するための管理シート作成。
    - 古橋氏が「今はフェーズ3の指示を出す」と判断しやすくする。
- **マニュアル・テンプレートの整備**
    - 加盟店向け運営マニュアルの各章テンプレート。
    - クライアントに「ここを埋めてください」と言える状態の雛形。

## 2. 【加盟店開発】に関する依頼事項
加盟店開発は「説得力のある資料」が武器です。

- **加盟検討者向け提案資料のブラッシュアップ**
    - 現在手元にある資料を、より成約率の高い構成へ改善。
- **リード（見込み客）管理フローの図解**
    - 問い合わせから契約までのステップを可視化。
    - CSが「次はどの資料を送るべきか」を迷わないためのフローチャート。

## 3. 【CS指示出しの仕組み化】への協力依頼
古橋氏の負担を減らすための「指示のテンプレート化」を依頼します。

- **指示用共通言語の策定**
    - 「〇〇の件」で通じるような、タスク名の標準化。
- **振り返り・ナレッジ蓄積の仕組み作成**
    - 過去のCSへの指示内容をアーカイブし、似た案件で使い回せる「指示出し事典」の作成。

3. 「次回のミーティングまでに、標準的な進捗管理シートのドラフトを作ってもらえるか？」
