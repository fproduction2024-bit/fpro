# 書店主軸・ゼンタングル提携拡大構想

## 📋 概要

**株式会社タングルシード**が展開するゼンタングル（マインドフルネスアート）事業の拡大戦略として、**書店を主軸にした全国定期ワークショップ＋スターターセット販売**モデルを構築する。

### 核心メッセージ
- **棚が動く＝講師が続く＝救われる人が増える**
- 講師は「教える人」ではなく、「回復の場をつくる人」
- 書店は会場ではなく、コミュニティの「家」

---

## 🎯 事業モデル

### 基本構造
```
参加者 → 店内ミニWS（月1回） → LINE登録 → 自宅体験講座 → 年間講座
                ↓
         スターターセット購入（書店売上）
```

### 参加費設定
- **2,200〜3,300円**（60分ワークショップ）

### 販売商品（2SKU戦略）
1. **入門セット**：衝動買い・レジ横向き（1,100〜1,650円想定）
   - ペン（サクラクレパス ピグマ）
   - タイル/はがき
   - 鉛筆・消しゴム
   - ミニガイド

2. **体験会推奨セット**：イベント連動（2,200〜3,300円想定）
   - ペン複数本
   - タイル複数枚
   - 道具セット
   - 簡易レシピ

## 👥 3者のメリット設計

### 1. 講師（100名以上のネットワーク）

#### 得られる価値
- **定期案件**：書店という「ホーム」で月1開催ができる
- **集客負担の軽減**：店頭告知・テンプレート提供
- **道具の統一**：推奨品（ピグマ）で初心者が迷わない
- **誇りと意義**：「文化の担い手」「回復の場をつくる人」としての社会的役割
- **評価制度**：品質基準とレビューによる昇給機会

#### 懸念と解消策
- **懸念**：品質ばらつき、クレーム対応、在庫負担
- **解消**：
  - 60分/15分の標準レシピ提供
  - 禁止事項明確化（営業、政治宗教、誹謗中傷）
  - 委託在庫・小ロット運用
  - 模擬審査・レビュー制度

### 2. 書店

#### 得られる価値
- **来店動機の創出**：月1定期イベントで習慣化
- **棚売上**：文具（ペン・紙）＋関連書籍の連動販売
- **滞在時間増加**：体験型コンテンツによる店舗価値向上
- **コミュニティの核**：地域文化の拠点化

#### 懸念と解消策
- **懸念**：運営負荷、クレーム、スペース占有
- **解消**：
  - 講師派遣と運営代行
  - 15〜30分コンパクト運営（非繁忙帯活用）
  - 最小什器持込
  - 賠償責任保険・規約整備

### 3. 株式会社タングルシード

#### 得られる価値
- **LINEリード獲得**：QR導線で自動集客
- **LTV最大化**：体験→年間講座のCVR向上
- **ブランド露出**：全国分散での認知拡大
- **データ資産**：ユーザー行動・購買データの蓄積

#### 懸念と解消策
- **懸念**：講師運用コスト、品質事故、在庫リスク
- **解消**：
  - 運用OS（標準レシピ・評価・配置最適化）
  - 品質審査・NPSトラッキング
  - 委託/小ロット運用

## 📚 なぜ書店を主軸にするのか

### 提携先比較

| 提携先 | 導入速度 | 継続性 | 商材連動 | 決裁 |
|--------|----------|--------|----------|------|
| **書店** | ◎ | ◎ | ◎（文具・紙） | 軽め |
| 企業 | △ | △（単発化） | × | 重い |
| 自治体 | ○ | ○（年度依存） | × | 中程度 |

### 結論
**店舗網 × 棚連動 × 低工数**で「継続導線」を作れる書店が最適

## 🎨 サクラクレパスとの協業シナリオ

### 背景
- ゼンタングル推奨ペン：**サクラクレパス「ピグマ」**
- 弊社オンライン講座で道具セット販売実績あり
- サクラクレパスは共同企画/OEMの受け皿を持つ

### 協業ステップ

#### Phase 1：自社で勝ちパターン作成（最初の90日）
1. 3店舗でパイロット実施
2. 棚回転・イベント稼働・再来店データ取得
3. 成功事例パッケージ化

#### Phase 2：共同提案（90日以降）
1. 数字を持ってサクラクレパスへ営業
2. 2SKU共同開発・監修
3. 公式SNS/店頭イベント連動
4. 共同販促（POP・什器・ノベルティ）

#### Phase 3：面展開
- 丸善ジュンク堂
- TSUTAYA/蔦屋書店
- 紀伊國屋書店
- 有隣堂 など大手書店チェーンへ横展開

## 📊 KPI設計

### 主要指標

#### 講師KPI
- 月間開催数
- 平均参加者数
- 再来店率
- 次回予約率
- 満足度（NPS）
- 紹介数

#### 書店KPI
- 契約店舗数
- 2SKU販売数/開催
- 関連文具・紙売上前年差
- イベント参加者数
- 滞在時間増加

#### 弊社KPI
- LINE登録数
- 登録→無料体験完了率
- 体験→年間講座CVR
- 継続率（解約抑制）

## 🛡️ ガバナンスと運用OS

### 品質基準
- 認定講師制度・模擬授業審査
- 年次更新
- NPSトラッキング

### 運用標準
- 15分レシピ/60分レシピ
- 必要物品チェックリスト
- 時間割テンプレート
- 什器最小化
- 会計動線設計

### 禁止事項
- 医療/効能表現
- 個人連絡先回収
- 無断撮影
- 強い営業・勧誘
- 政治・宗教活動
- 誹謗中傷

### 安全管理
- 賠償責任保険加入
- インシデント報告フロー
- 天候・来客変動時の運用ルール

### 店頭導線
- POP/QR設置ガイド
- スタッフ用トークスクリプト
- レジ横カード配置図

## ⚠️ リスクと対策

| リスク | 対策 |
|--------|------|
| 集客割れ | 事前予約枠＋店頭告知テンプレ＋SNS連携 |
| クレーム/品質問題 | 講師審査・NPS可視化・代替対応プロトコル |
| 講師偏在 | エリアプール化・交通費規定・オンライン代替 |
| 在庫過不足 | 小ロット・短納期補充・委託運用 |
| 外部要因（天候/改装） | 予備日・代替会場・オンライン振替 |
| ブランド毀損 | SNS/写真ポリシー・対応フロー明確化 |

## 💰 収益分配モデル（案）

### プランA：売上按分（推奨）
- 書店：30〜50%（会場提供・レジ対応・告知協力）
- 講師/運営：残り
- **メリット**：書店が「売上が立つ」実感を得やすい

### プランB：場所代固定＋物販は書店100%
- 書店：場所代（5,000〜15,000円/回）＋文具/書籍売上
- 運営：参加費収入
- **メリット**：書店が「商品が動く」実感を得やすい

## 📌 重要な設計思想

### 1. 講師を主役にする
講師は「教える人」ではなく、**「失敗のない文化を社会に増やす人」**「**自己肯定感が上がる場を、街に増やす人**」

### 2. 書店は「会場」ではなく「家」
コミュニティが毎月帰ってこられる場所

### 3. 弊社の売上は「自然増」でよい
書店の売上（棚）を主目的にすることで、枠が残り、露出が積み上がり、結果的にLINE導線で弊社売上も増える

### 4. 全国分散×講師ネットワーク
講師100名以上という資産を「光の網」として、全国に小さなコミュニティを増殖させる

### 5. 上達導線がコミュニティを育てる
ワークショップ（入口）→ 上達（LINE・オンライン講座）→ コミュニティ（継続・紹介）
