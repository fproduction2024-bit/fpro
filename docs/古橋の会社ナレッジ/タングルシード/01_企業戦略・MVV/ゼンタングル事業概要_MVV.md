# ゼンタングル事業概要 - MVV分析

## 📋 基本情報

### 事業名
**みどりタングルサロン**

### 講師
- **古橋みどり**
- 京都市立芸術大学日本画卒業
- 国内外で展覧会多数
- フランスでの書籍出版実績
- 2018年 ゼンタングル®講師資格CZT取得

## 🎯 MVV (Mission, Vision, Values)

### Mission（ミッション）- 私たちの使命

**「つらい状況にある人に楽しい時間を持ってもらい、自信を持って生きてもらう」**

- アートの力で、誰にも病気にも奪い取られない豊かさを心の中に広げる
- ゼンタングルを通じて、人生を変える体験を提供する
- 絵心がない人でも、美しい絵が描ける喜びを届ける

### Vision（ビジョン）- 目指す未来

**「ゼンタングルで世界中の人々が繋がり、ポジティブなコミュニティを形成する」**

- 世界中のゼンタングルコミュニティとの繋がり
- SNSを通じた国際的な交流の促進
- 「間違い」「失敗」「順位づけ」のない、認め合う文化の創造
- 居心地の良いコミュニティの拡大

### Values（価値観）- 大切にしていること

1. **癒しと自己肯定感の向上**
   - セラピー効果による気持ちのスッキリ感
   - 自己肯定感が上がり、前向きに活動できるようになる
   - もやもやした気持ちや不安を忘れられる

2. **誰でも参加できる包摂性**
   - 絵心がなくても大丈夫
   - 間違いや失敗がない
   - お互いを認め合い、褒め合う文化

3. **マインドフルネスと集中**
   - 時間を忘れるほどの集中体験
   - 瞑想・呼吸法以上のマインドフルネス効果
   - 描いている間の気持ちの楽さ

4. **コミュニティとつながり**
   - 寛大で認め合うコミュニティ
   - 世界中の人とポジティブなコミュニケーション
   - SNSでのいいねやコメントによる承認

## 🎓 提供サービス

### みどりタングルサロン（サブスクリプション型）

#### 提供内容
1. **月3回配信のゼンタングル動画講座**
   - 1〜3時間程度の講座
   - 予測しないで描くライブ形式
   - 過去講座も会員サイトで視聴可能

2. **コミュニティ**
   - Facebookグループ
   - LINEでのコンテンツ配信
   - YouTubeライブ配信

3. **お道具セット**
   - 初回に必要な道具一式を提供
   - 追加の紙は各自で用意（画用紙、100均の紙でもOK）

#### 料金
- **年間一括払い**: 59,400円（税込）
  - お道具セット（8,200円相当）無料プレゼント
- **月額分割払い**: 4,950円/月（税込）
  - 初回のみ 13,150円（月額4,950円 + お道具セット8,200円）
  - 2ヶ月目以降 4,950円/月

#### 参加の流れ
1. 参加ボタンを押す
2. LINEでお支払いページから手続き
3. お道具セットの送付先を入力
4. Facebookグループ・会員サイトに参加
5. ゼンタングルを楽しむ

### 無料プレゼント講座

LINE友だち登録で以下の講座を無料提供:
1. **初心者向け講座** - スッと描ける動画クラス
2. **深い集中講座** - セラピー効果を感じられるタングル
3. **応用講座** - メッセージカードの縁飾り

### カリキュラム詳細（1年間プログラム）

#### 前半6ヶ月：ゼンタングルプライマー Vol.1
- 創始者リック・ロバーツ氏とマリア・トーマス氏のメソッド解説
- ゼンタングルの8つのステップ
- 基本的なタングル（パターン）の習得
- 月2回：プライマークラス
- 月1回：ゼンタングルインスパイアードアート

#### 中盤2ヶ月：タングルスピナー制作
- 60個の新しいタングルを学習
- 大型作品の完成（額装可能）
- レジェンド（運任せで描く）技法の習得

#### 後半4ヶ月：カラータイルクラス
- グレー、茶色、黒の紙への描き方
- 白ペン、金ペンの使用技術
- 技術面の深掘り

## 📖 創業ストーリー - 講師の原体験

### がん闘病からの気づき

古橋みどり講師は、肺腺がんステージ3B（5年生存率20%）の告知を受け、8回の入退院を経験。

#### 入院中の発見
- 患者同士の暗い話題や悲観的な情報に囲まれる中、絵を描くことで気持ちが楽になった
- 描いている間は落ち込むことなく治療に向き合えた
- 病院という隔離された環境でも、アートが「自分が自分でいられる世界」を心の中に広げてくれた
- **誰にも、病気にも奪い取られない豊かさ**がアートにはある

#### 使命の発見
> 「つらい状況にある方、みんなにアートの力を手にして明るい気持ちで生きていってほしい」

ゼンタングルを手ほどきした人から「これのおかげで人生が変わった」と喜ばれ、生きる意味を実感。

## 🌍 コミュニティの特徴

### ゼンタングルコミュニティの文化

1. **寛大で認め合う人が多数**
   - お互いを褒め合う
   - ポジティブなコミュニケーション
   - 「びっくりするくらい良い人ばっかり」（生徒の声）

2. **SNSでの繋がり**
   - #zentangle をつけて投稿
   - 世界中からいいねやコメント
   - フォロワーが増えやすい

3. **変化が見える**
   - 「お店に来るときの顔と、描いて帰るときの顔が全然違う」（カフェ店長の声）
   - 一緒に描く喜び、認め合う心地よさが人を安心させる

### 受講者の声・効果

1. **「人生が変わりました」**
   - 絵を描いて「すごいね」と言われたことがなかった
   - ゼンタングルで自信がつき、性格が前向きに変化
   - 講師資格を取得し、現在は教える側に

2. **「世界が変わりました」**
   - SNS（#zentangle）で世界中からいいねやコメント
   - 地域を超えた繋がりができた
   - ポジティブな声掛けで気持ちが外に向かって楽になった

3. **「顔が変わりました」（カフェ店長の声）**
   - クラス前と後で表情が全く違う
   - 2年間の継続で外見も変化（ダイエット成功、表情が明るく）
   - 自信がつき、手作り品販売を開始

4. **心理的効果**
   - 不安な気持ちが減り、薬なしで眠れるようになった
   - 病院の待ち時間も不安なく過ごせる
   - 仕事が終わるのが楽しみ、怒る回数が減った
   - 腰痛を忘れるほどの集中
