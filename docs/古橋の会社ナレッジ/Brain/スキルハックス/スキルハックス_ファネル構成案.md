# スキルハックス版ファネル動画 ── 全体構成マップ

> **商品**: Skill Hacks（全講座学び放題 + 伴走支援）
> **価格**: 44万円（税込） = 1日約500円
> **ポジショニング**: 「安くて手厚い」（市場の空白地帯）
> **全体コンセプト**: 「新しいことを勉強するのをやめて、今すぐ稼ぎ始めませんか？」
> **方程式**: Skill + ○○ = Skill Hacks

## 各話のストーリーアーク

```
第一話: 気づき ─────────────────────────────────
  「ノウハウを知ること自体に価値はない」
  「スキルを体系的に習得すること」だけが現状を変える唯一の方法
  → 成果が出る人と出ない人の違いは？ → 次回へ

第二話: 理解 ─────────────────────────────────
  成果を出す人が持つ3つの武器
  ① コアスキルだけに集中  ② 完璧前にお金をもらう  ③ 環境
  迫さんの泥臭い初案件ストーリー
  キャッシュフローの教育（労働収入→資産→自動化）
  → でもこの3つを全部自力で揃えるのは至難 → 次回へ

第三話: 確信 ─────────────────────────────────
  3つの武器を全て実現する仕組み＝スキルハックス
  市場の2軸マトリクス → 「安くて手厚い」という空白ポジション
  3つの柱: 全講座学び放題 / 伴走支援 / コミュニティ
  2倍返金保証 / 教育への想い / 向いている人・いない人
  → 詳細・お申し込みはセールス動画へ

VSL: 決断 ─────────────────────────────────
  理想の未来の追体験（半年後の自分）
  不安を1つずつ解消
  全講座・サポートの詳細説明
  受講生の声・実績データ
  価格正当化 → 競合比較 → 1日500円
  限定特典 → リスクリバーサル → FAQ
  最終クロージング → 15分面談オファー
```

## 全体を貫くキーメッセージ

1. **「ノウハウを知ること」に価値はない。「体系的なスキル習得」だけが現状を変える**
2. **スキルだけでは足りない。スキル＋○○（環境・サポート・仲間）＝スキルハックス**
3. **世の中の知識は高すぎる。コーヒー1杯分で人生が好転する社会であってほしい**
4. **自信がついたら行動するのではなく、行動するから自信がつく**
5. **失敗は存在しない。唯一の失敗は何もしないこと**
