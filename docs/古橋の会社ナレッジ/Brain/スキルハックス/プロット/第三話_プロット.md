# 第三話 プロット：スキルを身につけたあなたが、確実にお金に変える方法

> **尺の目安**: 25〜30分
> **本話の位置づけ**: コンテンツの最終話。ここでセールス動画（VSL）への接続を行う
> **核心メッセージ**: スキル習得後にも収益化を阻む「見えない壁（＝小石）」が存在する。それを全て解決する仕組みが「スキルハックス」

---

## 1. 導入：前回までの振り返り

### 1-1. シリーズ全体の復習
- **第一話**: スキルを身につけて副業で稼ぐという道がある。成果が出る人と出ない人の違いは「枝葉のノウハウを集めるか」「幹のスキルに集中するか」
- **第二話**: 最短で結果を出す人が持っている3つの武器
  - ① コアスキルだけに集中
  - ② 技術が習熟する前にお金をもらう
  - ③ 挫折させない環境（サポート・仲間）

### 1-2. 本動画の目的
- 前回までで「何を学ぶか」「どう行動するか」は分かった
- **では、その3つの武器を全て手に入れる方法は？**
- 今回は、スキルが身についても多くの人がぶつかる「**最後の壁**」の正体を明かし、それを全て解決する仕組みを公開する
- **後半にはこの動画シリーズを見てくださった方限定の特別なお知らせがある**

## 2. 視聴者の不安の代弁

### 2-1. 共感パート
- 「ここまで聞いて、すごく納得した。でも、本当に自分にできるのか？」
- 「周りに副業で稼いでいる人がいない。成功イメージが湧かない」
- 「情報商材やオンラインスクールって詐欺じゃないの？」
- 「投資するお金、ムダになったらどうしよう」
- → 「その気持ち、痛いほどよく分かります。あの273円の1年間を過ごしていた僕自身が、まさにそうでした」

### 2-2. 共感からの転換
- しかし、あの時の僕にもし「正しい方法」と「正しい環境」があったら？
- 1年間の遠回りをせずに、最初の1ヶ月で結果を出せていた
- → 先を行く人の方法をトレースすれば、最短で結果が出る。これは才能ではなく**仕組み**の問題

## 3. 常識破壊：「一般的な教材」と「スキルハックス」の根本的な違い

### 3-1. 市場の現状（2軸マトリクス）
| | **サポート手厚い** | **売り切り** |
|---|---|---|
| **高額** | おさるさん・レベラ等（88万円〜）| 高額情報商材（50万〜100万円）|
| **低額** | **← ここがスキルハックス** | Udemy・note等 |

- **高くて手厚い**: 創業者本人が直接指導、質問に2分で返信。しかし88万円は高すぎないか？
  - しかも、その実態の多くは創業者本人ではなくスタッフが対応している
- **安くて売り切り**: 安いが結果が出ない。自己管理能力の高い1割の人しか成功できない。○○ハックスの過去もここだった
- **高くて売り切り**: 最悪のパターン。高額なのにサポートなし

### 3-2. スキルハックスの独自ポジション：「安くて手厚い」
- ○○ハックスシリーズは「安くて売り切り」に近かった → 結果を出せない学習者がいたという**反省**
- **だから作った**: 同等以上の手厚いサポートを、圧倒的に安い価格で提供するサービス
- **信念**: 「世の中の知識は高すぎる。コーヒー一杯分くらいの投資で人生が好転する社会であってほしい」
- **覚悟**: この価格設定は市場の既存プレイヤーすべてを敵に回す。しかし「初心者を対象とした0→1サポートなのに88万円もするのはおかしい」と、僕は堂々と言いたい

## 4. スキルハックスの3つの柱

### 4-1. 柱①：全講座学び放題（○○ハックスシリーズ全部 ＋ Brain講座全部）

#### スキル系（ゼロから案件獲得まで）
- **ProgrammingHacks**: Webプログラミングの基礎。脱初心者を最短で
- **MovieHacks**: YouTubeに特化した動画編集。案件獲得支援コース付き
- **DesignHacks**: デザイン基礎〜Photoshop/Illustrator。仕事の取り方まで
- **WritingHacks**: 高単価ライティング＋単価交渉＋案件獲得

#### ビジネス・マーケティング系（稼いだ後を加速させる）
- **ビジネスモデルマスター講座**: 全ての商売に共通する仕組みの作り方。迫佑樹が手掛ける実業を参考に16本の動画
- **Xマーケティングマスター講座**: フォロワー増加→収益化の10テンプレ
- **LINEマーケティングマスター講座**: 顧客との濃密な関係構築で売上飛躍
- **YouTube裏ノウハウ全公開**
- **Instagram運営ノウハウ全公開**

#### ポイント
- 単品で買うと合計78.8万円分以上 → それが全て学び放題
- 「自分に合ったスキルが分からない」→ 全部試せるから、やりながら適性が分かる
- スキルの**掛け合わせ**で市場価値が爆発的に上がる（ライティング×デザイン、動画編集×マーケティング等）

### 4-2. 柱②：伴走支援（サポート体制）── 「小石を全部なくす」環境
- **初回プランニング面談**: あなた専用のロードマップ作成（何を・いつまでに・どの順番で学ぶか）
- **オンライン作業会（もくもく会）**: 仲間と一緒にリアルタイムで作業。孤独感ゼロ
- **個別面談**: つまずきポイントの解消、メンタルサポート
- **無制限質問対応**: 分からないことが出てきたら即解決
- → 2話で話した「小石」（応募文の書き方、クライアント対応、お金をもらう怖さ…）を**全部なくす**
- → **「何をすればいいか分からない」状態が、物理的に発生しない設計**

### 4-3. 柱③：コミュニティとマインド教育
- **受講生同士の仕事発注**: 先輩受講生がチームに入れてくれることも
- **成功事例の共有**: 「○○さんが月30万達成！」→ 自分もいける、というリアルな実感
- **稼いだ先の幸福まで設計**: 人生攻略ロードマップ（労働収入→資産構築→自動化）
- → 単なる「講座」ではなく「**一生高め合える仲間**」

## 5. 黄金パターンの提示

### 5-1. ○○ハックスが10年×3万人で見つけた「黄金パターン」
- ① **必須スキルだけ**を最短で習得する（テンプレ活用・過学習排除）
- ② **技術が習熟する前に案件を取りに行く**（クラウドソーシング営業テンプレ付き）
- ③ つまずいたら**すぐに質問・相談**（小石を即除去）
- ④ 案件をこなしながら**実務でスキルUP**（正のスパイラル）
- ⑤ スキルの掛け合わせ・単価交渉で**収入を加速**
- ⑥ 稼いだお金で**資産を構築**（メディア・事業化）

### 5-2. この黄金パターンを回すのに必要な要素
- **体系化された最短カリキュラム** ← ○○ハックスシリーズ
- **つまずきポイントを全て潰すサポート体制** ← 伴走支援
- **成功している先輩・仲間** ← コミュニティ
- → **この3つが全て揃っているのが「スキルハックス」**

## 6. 2倍返金保証

### 6-1. 保証の内容
- 決められた課題（日報・成果報告・SNS発信）をこなしても受講料分すら稼げなかった場合
- → **未回収額の2倍を返金**
- 「受講料44万円のうち、30万しか稼げなかった → 残り14万の2倍＝28万円キャッシュバック」

### 6-2. なぜこの保証ができるのか
- **カリキュラムへの絶対的な自信の裏返し**
  - 3万人以上のデータに基づいた黄金パターン
  - やれば必ず結果が出る仕組み
- **リスクはゼロ**: やった人が損をすることは構造的にあり得ない
- 「万が一合わないと感じた場合のために1ヶ月以内の全額返金保証も検討」

## 8. 参加の判断基準

### 8-1. 向いている人
- 自分のスキルで稼ぐ経験を積みたい人
- 場所と時間に縛られない働き方を実現したい人
- コツコツ積み上げることが苦痛じゃない人
- 自分の成長に投資する覚悟がある人
- 素直に行動できる人

### 8-2. 向いていない人（＝デメリットの提示）
- インプットだけで行動しない人
- 他責思考で言い訳をする人
- 楽して稼ぎたいだけの人
- ノウハウを悪用して人を騙そうとする人
- **既に別事業で成功している中上級者がコンサルティングを求めるケース** → スキルハックスは**0→1達成に特化**しているため対応できない
- → この「何でも屋ではない」という**制約があるからこそ、この価格が実現できている**
