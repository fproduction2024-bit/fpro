## 1. 導入：3本の動画の要点再掲

### 1-1. シリーズ総括
- **第一話**: スキルを身につけて副業で稼ぐのが最もリスクが低い道。ノウハウを「知る」のではなく「体系立てられたスキルを習得する」ことだけが価値を持つ

- **第三話**: 3つの武器を全て実現する「スキルハックス」の全貌。安くて手厚いという独自ポジション、全講座学び放題＋伴走支援＋コミュニティの3つの柱

- **参加するもしないもあなたの自由**。ただし判断に必要な情報は全て提供する

## 3. 視聴者の不安を1つずつ解消

### 3-1. 「本当に自分にもできるのか？」
- 迫さん自身の体験: 273円の1年間 → 仕組みと環境を変えた瞬間に結果が変わった
- 受講生3万人のうち、最初から才能があった人は一人もいない
- → **できるかどうかではなく、やるかやらないかだけ**

### 3-2. 「情報商材や詐欺じゃないの？」
- Brain公式メディア運営の透明性
- テレビ出演・大学講義という対外的信頼
- **デメリットを正直に提示している**時点で、典型的な情報商材とは構造が違う
  - 0→1特化であること、中上級者には不向きであること → 正直に伝えている

### 3-3. 「周りに副業で成功している人がいない」
- だからこそコミュニティがある
- 一人でやるから不安。仲間がいれば「普通のこと」になる
- 受講生の成功事例を見せる（スライド＋可能ならインタビュー動画）

### 3-4. 「お金がもったいない」
- 44万円 = 1日約500円 = コーヒー1杯分
- しかも2倍返金保証 → 損する構造が物理的にない
- 88万円の高額スクールと比較すれば半額以下

### 3-5. 「時間がない」
- 本業と両立しているメンバーが大多数
- カリキュラムはオンライン完結、自分のペースで進められる
- 伴走支援でタイムマネジメントもサポート

## 4. スキルハックスの詳細解説（第三話の深掘り）

### 4-1. 全講座の個別解説
※第三話で概要を伝えたものを、ここでは各講座の**具体的なカリキュラム内容**と**受講後にできるようになること**をより詳しく解説

- **ProgrammingHacks**: 基礎的なサイト構築、Webアプリ開発の基本。卒業後はクラウドソーシングでコーディング案件を受注可能
- **MovieHacks**: YouTube特化の動画編集。テロップ・カット・BGM・音入れ・画像挿入の必須5項目を最短で習得。案件獲得支援コースで営業まで一貫サポート
- **DesignHacks**: バナー・LP・SNS画像のデザイン。Photoshop/Illustratorの実務操作。名刺やチラシの制作案件にも対応
- **WritingHacks**: SEOライティング＋単価交渉。月額1万円/記事 → 文字単価3円以上のライターへ
- **ビジネスモデルマスター**: 事業設計の本質16本
- **Xマーケティングマスター**: フォロワー運用10テンプレ
- **LINEマーケティングマスター**: マガジン配信で売上拡大
- **YouTube / Instagram**: メディア運営のノウハウ

### 4-2. 伴走支援の詳細
- **初回プランニング面談の流れ**:
  - ヒアリング → あなたの現状・目標を把握
  - ロードマップ作成 → 何を・いつまでに・どの順番で
  - 最初の1週間のアクションプランを一緒に策定
- **もくもく会（オンライン作業会）**の雰囲気
  - 作業BGMが流れる中、カメラONで一緒に作業
  - 分からないことはリアルタイムで質問
  - 終了後にその日の進捗を共有
- **個別面談（つまずきポイント解消）**
  - 「案件に応募する勇気が出ない」→ 一緒に提案文を添削
  - 「クライアントから修正依頼が来たけどどう返す？」→ メッセージの書き方を一緒に練る
  - **「何をすればいいか分からない」が物理的に発生しない設計**

### 4-3. コミュニティの実態
- 受講生同士の仕事発注の事例
- 成功事例の日々の共有（「今日初案件取れました！」→ みんなで祝福）
- 先輩受講生がキャリア相談に乗ってくれる

### 6-2. 価格の提示
- **通常価格**: ※要決定
- **本日限定価格**: **44万円（税込）**
- **1日あたりに換算するとたった500円。コーヒー1杯分**

### 6-3. 競合比較
- 高額スクールA: 88万円（サポートはスタッフ対応）
- 高額コンサルB: 100万円以上（創業者が直接対応するが、受講生は少数）
