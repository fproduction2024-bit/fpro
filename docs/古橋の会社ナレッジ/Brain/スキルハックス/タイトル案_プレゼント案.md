# スキルハックス ファネル動画 ── タイトル案 & プレゼント案

## 全体構成（確定）

```
第一話（5〜10分）──→ 第二話（25〜30分）──→ 第三話（25〜30分）──→ VSL（約1時間）
   導入・気づき          教育・理解            確信・オファー導入        セールス・決断
                                                                    ↓
                                                              【プレゼント配布】
```

## プレゼント案

### 配布タイミングと目的

```
第一話 → 第二話 → 第三話 → VSL視聴 → 【プレゼント配布】→ 購入検討
                                              ↑
                                      「行動のきっかけ」を作る
                                      ＋購入への自然な導線
```

- **配布条件**: VSL（セールス動画）を最後まで視聴した人限定
- **目的**: ① 視聴完了率UP ② プレゼント自体が「スキルハックスの体験版」になる ③ 使ってみて「伴走支援が欲しい」と自然に思わせる

---

### 案1：副業スキルマッチング診断シート
**形式**: PDF（4ページ）またはスプレッドシート

| 項目 | 内容 |
|------|------|
| **中身** | 10個の質問に答えると、自分に最適なスキル（動画編集/プログラミング/ライティング/デザイン）が分かる |
| **付録** | 各スキルの「市場規模・平均単価・習得期間・向いている性格」比較表 |
| **制作コスト** | 低（テキスト＋簡単な表組み） |
| **購入導線** | 「自分に合うスキルは分かった → でも何から始めれば？ → スキルハックスなら全部学べる」 |

> **強み**: 「何を学べばいいか分からない」という最大の行動障壁を除去
> **弱み**: ツール感は薄い。情報寄り

---

### 案2：副業ゼロイチ 90日ロードマップ
**形式**: PDF（8〜10ページ）

| 項目 | 内容 |
|------|------|
| **中身** | Day 1〜7: スキル選定 → Day 8〜21: コアスキル習得 → Day 22〜30: 初案件獲得 → Day 31〜60: 安定受注 → Day 61〜90: 月10万円達成 |
| **付録** | 各フェーズの具体アクション＋チェックリスト |
| **制作コスト** | 中（デザインを入れると見栄えが良い） |
| **購入導線** | 「全体像は見えた → でも一人でやると小石に躓く → 伴走支援で確実にやりたい」 |

> **強み**: ゴールまでの全体像が一目で分かる。地図としての実用性が高い
> **弱み**: 情報としては濃いが「ツール」感がやや弱い

---

### 案3：初案件獲得テンプレート集
**形式**: PDF（12〜15ページ）または Notionテンプレ

| 項目 | 内容 |
|------|------|
| **テンプレ①** | クラウドソーシング用プロフィール文（職種別4パターン：動画編集/プログラミング/ライティング/デザイン） |
| **テンプレ②** | 提案文テンプレ（初心者がコピペできるレベル＋カスタマイズ箇所の解説） |
| **テンプレ③** | 「相場マイナス1,000円」価格戦略の具体例＋理由の書き方 |
| **テンプレ④** | クライアントへの初回メッセージ・修正対応・納品メッセージのテンプレ |
| **制作コスト** | 中〜高（実際に使えるクオリティが必要） |
| **購入導線** | 「テンプレで案件取れそう → 実際やると小石に躓く → 伴走支援が欲しい」 |

> **強み**: 即座に使える実用ツール。「これだけで案件取れるかも」という期待感
> **弱み**: テンプレの質が低いとブランド毀損。制作にやや手間がかかる

---

### 案4：スキルチェックリスト
**形式**: PDF（6〜8ページ）またはスプレッドシート

| 項目 | 内容 |
|------|------|
| **中身** | 職種別（動画編集/プログラミング/ライティング/デザイン）の「案件獲得に必要な最低限スキル」一覧 |
| **仕組み** | 各スキルに ☐ チェックボックス → 全部埋まったら即営業OK |
| **付録** | 埋まらない項目の横に「○○ハックスの第○章で学べる」と記載 |
| **制作コスト** | 低（リスト＋チェックボックス） |
| **購入導線** | 「チェックが埋まらない → スキルハックスで全部埋まる → 買おう」 |

> **強み**: ミーティングで合意済み。動画シリーズ内で繰り返し予告しているので視聴者の期待がある
> **弱み**: 単体だと地味。テンプレ集とセットにするとパワーアップ

---

### 案5：副業収支シミュレーター
**形式**: Googleスプレッドシート（コピーして使用）

| 項目 | 内容 |
|------|------|
| **入力項目** | スキル種別 / 想定単価 / 月の稼働時間 / 経費 |
| **自動計算** | 月収 / 年収 / 時給 / 手取り額 |
| **可視化** | キャッシュフロー4つの箱（収入・支出・資産・負債）がグラフで表示 |
| **シミュレーション** | 「労働収入→資産構築→自動化」の各フェーズ到達予測時期 |
| **制作コスト** | 高（スプレッドシートの関数・グラフ設計が必要） |
| **購入導線** | 「副業したらいくら稼げるか」が具体的に見える → 現実味 → 始めたい |

> **強み**: ツール感が最も強い。繰り返し使える。迫さんのキャッシュフロー教育と連動
> **弱み**: 制作コスト高。スプレッドシートに不慣れな層は使いこなせない可能性

## おすすめ組み合わせ

### 推奨：案4 + 案3 のセット配布
**「スキルチェックリスト ＆ 初案件獲得テンプレート集」**

```
スキルチェックリストでチェック
        ↓
  全部埋まった → テンプレ集で即営業
  埋まらない → スキルハックスで学ぶ
        ↓
  テンプレで営業開始
        ↓
  小石に躓く（応募の怖さ、クライアント対応…）
        ↓
  「伴走支援があったら…」→ スキルハックス購入
```

**理由**:
- チェックリストは動画内で繰り返し予告済み → 視聴者の期待に応える
- テンプレ集は即使える実用ツール → 「もらって損した」感がゼロ
- 2つが補完関係にある（診断→行動）
- 制作コストも中程度で現実的

### 次点：案4 + 案5 のセット配布
**「スキルチェックリスト ＆ 副業収支シミュレーター」**

- シミュレーターで「稼げる金額」を可視化 → チェックリストで「あと何を学べばいいか」を確認
- より「ツール感」が強いが、制作コストが高い
