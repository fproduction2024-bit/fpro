# 受講生ヒアリング 質問テンプレート

> **目的**: 台本（第二話・第四話VSL）に挿入する受講生事例を収集する
> **所要時間**: 約15〜20分
> **実施方法**: Zoom / 対面 / テキスト回答

---

## 事前準備

- [ ] ヒアリング対象者をリストアップ（3〜5名）
- [ ] 事前に「台本に使う可能性がある」旨を伝え、掲載許可を取る
- [ ] 録音・録画の許可を取る（Zoomの場合）

---

## 質問項目

### 1. 基本情報

- お名前（ニックネーム可）
- 年齢層（20代 / 30代 / 40代〜）
- 受講前の職業・状況（会社員 / 主婦 / 学生 / フリーター 等）
- 家庭環境（子育て中 / 共働き / 単身 等）

### 2. 受講前の状態

- 副業を始めようと思ったきっかけは？
- 受講前に感じていた不安や悩みは？
- 以前に他の教材やスクールを試したことは？ → その結果は？
- 「自分にもできるのか」という不安はありましたか？

### 3. 受講の決め手

- スキルハックスを知ったきっかけは？
- 参加を決めた一番の理由は？
- 価格や返金保証は判断にどう影響しましたか？

### 4. 受講中の体験

- 最初に学んだスキルは何ですか？
- 学習ペースはどのくらいでしたか？（1日何時間、何ヶ月で完了 等）
- 本業・家事との両立で工夫したことは？
- **つまずいたポイント**とその時どうしたか？
- 伴走支援（個別面談・もくもく会等）は使いましたか？ → 具体的にどう役立った？
- コミュニティの雰囲気はどうでしたか？

### 5. 案件獲得の経験

- 初案件を取るまでに何件くらい応募しましたか？
- 初案件の内容と単価は？
- 初めてお金をもらった時の気持ちは？
- その後、リピートや紹介はありましたか？
- 現在の月収（副業収入）はどのくらいですか？

### 6. 変化と実感

- 受講前と今で、一番変わったことは？
- 「自信がついた」と感じた瞬間は？
- 家族や周囲の反応は？
- 働き方や生活はどう変わりましたか？

### 7. 台本用の一言（★重要）

以下のような「一言コメント」をいただけると台本に引用しやすいです：

- 「○○だった私が、△△できるようになった」
- 「あの時一歩踏み出して本当によかった」
- 「スキルハックスの○○があったから続けられた」

**※ 自然体の言葉でOKです。後でこちらで台本に合わせて調整します。**

## ヒアリング後のアクション

- [ ] 回答を整理して台本チームに共有
- [ ] 台本の該当箇所に事例を挿入（第二話 §5、第四話 §4・§6）
- [ ] 本人に掲載箇所の最終確認を取る

## 欲しい受講生タイプ（優先度順）

| 優先度 | タイプ | 理由 |
|--------|--------|------|
| ★★★ | 子育て中の主婦 | ターゲット層と重なる、共感度が高い |
| ★★★ | 完全未経験からゼロイチ突破 | 「自分にもできる」感を伝えやすい |
| ★★☆ | 本業ありの会社員 | 時間がない中での成功事例 |
| ★★☆ | 40代以上の方 | 年齢の壁を超えた事例 |
| ★☆☆ | 他スクール経験後にスキルハックスで成功 | 差別化の証明 |
