# スキルチェックリスト

> **使い方**: 各スキルの項目をチェックしていきましょう。全項目にチェックが付いたら、あなたはもう案件に応募できる状態です。埋まらない項目があれば、そこだけを集中的に学びましょう。
>
> Skill Hacks 受講生特典

---

## 🖥️ ProgrammingHacks（プログラミング）

### 基礎スキル
- [ ] HTML/CSSの基本構造を理解している
- [ ] レスポンシブデザイン（スマホ対応）が実装できる
- [ ] JavaScriptの基本文法（変数・関数・条件分岐・ループ）を理解している
- [ ] jQueryまたはVanilla JSでDOM操作ができる

### 実務スキル
- [ ] 既存のWebサイトのデザインを見て、同じものをコーディングできる
- [ ] WordPressの基本的なカスタマイズができる
- [ ] ブラウザの開発者ツール（DevTools）を使ってデバッグできる
- [ ] GitHubの基本操作（clone, commit, push）ができる

### 案件獲得の最低ライン
- [ ] LPコーディングのサンプルを1つ以上作成した
- [ ] ポートフォリオサイトを公開している

## 🎬 MovieHacks（動画編集）

- [ ] 書き出し設定（解像度・ビットレート）を理解している

### 案件獲得の最低ライン
- [ ] YouTube動画の編集サンプルを1本以上作成した
- [ ] ポートフォリオ（サンプル動画のリンク集）を用意している
