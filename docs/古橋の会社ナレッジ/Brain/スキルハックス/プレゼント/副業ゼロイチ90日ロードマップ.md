# 副業ゼロイチ 90日ロードマップ

> スキルゼロ・実績ゼロの状態から、90日間で「月10万円の副業収入」を目指すためのロードマップです。
> 各フェーズのアクションとチェックリストを順番にクリアしていきましょう。
>
> Skill Hacks 受講生特典

---

## 全体マップ

```
Phase 1（Day 1〜7）   スキル選定・環境準備
Phase 2（Day 8〜21）  コアスキル習得
Phase 3（Day 22〜30） 初案件獲得
Phase 4（Day 31〜60） 安定受注・スキルアップ
Phase 5（Day 61〜90） 月10万円達成・仕組み化
```

---

## Phase 1：スキル選定・環境準備（Day 1〜7）

### 目標
「何をやるか」を決めて、学習をスタートできる状態にする。

### Day 1〜2：自分に合うスキルを選ぶ

| アクション | 詳細 |
|-----------|------|
| 診断シートを使う | 「副業スキルマッチング診断シート」で自分のタイプを確認 |
| 市場調査をする | クラウドワークスで各スキルの案件数・相場を確認 |
| 1つに決める | 迷ったら伴走支援の初回面談で相談 |

### Day 3〜5：環境を整える

| アクション | 詳細 |
|-----------|------|
| ツールを準備 | 必要なソフトをインストール（Premiere Pro / Photoshop 等） |
| アカウント登録 | クラウドワークス・ランサーズに登録 |
| 学習環境 | 毎日の学習時間と場所を決める（例：21時〜23時） |

### Day 6〜7：カリキュラム開始

| アクション | 詳細 |
|-----------|------|
| 学習開始 | スキルハックスのカリキュラム（基礎編前半）に着手 |
| 学習記録 | 毎日の学習内容と気づきをメモ |

### Phase 1 チェックリスト
- [ ] 取り組むスキルを1つ決めた
- [ ] 必要なツール・ソフトをインストールした
- [ ] クラウドワークスに登録した
- [ ] 毎日の学習スケジュールを決めた
- [ ] カリキュラムの学習を開始した

---

## Phase 2：コアスキル習得（Day 8〜21）

### 目標
案件を受けられる「最低限のスキル」を身につける。

### Day 8〜14：基礎スキルを固める

| アクション | 詳細 |
|-----------|------|
| カリキュラム消化 | 基礎編を完了させる（1日2〜3時間目安） |
| 手を動かす | 見るだけでなく、必ず自分で再現する |
| 質問する | 分からないことがあったら即Q&Aサポートへ |

**スキル別の「基礎完了」ライン:**

| スキル | ここまでやればOK |
|-------|----------------|
| 動画編集 | テロップ・カット・BGM・SE・画像挿入の5つが一人でできる |
| デザイン | Photoshopでバナーを1枚作れる |
| プログラミング | HTML/CSSで簡単なページを1枚コーディングできる |
| ライティング | SEOを意識した3,000字の記事を1本書ける |

### Day 15〜18：サンプル作品を作る

| アクション | 詳細 |
|-----------|------|
| サンプル制作 | 実際の案件を想定した作品を制作 |
| 数の目安 | 動画編集：1本 / デザイン：バナー3枚 / プログラミング：LP1ページ / ライティング：記事2本 |
| フィードバック | コミュニティや伴走支援メンバーに意見をもらう |

### Day 19〜21：ポートフォリオを整える

| アクション | 詳細 |
|-----------|------|
| まとめる | サンプルをGoogleドライブ / Notionにまとめる |
| URLを発行 | 共有リンクを作成（提案文に貼れるように） |
| 見せ方の工夫 | 制作意図や工夫ポイントを一言添える |

### Phase 2 チェックリスト
- [ ] カリキュラムの基礎編を完了した
- [ ] 基本操作が一人でできるようになった
- [ ] サンプル作品を規定数制作した
- [ ] ポートフォリオURLを用意した
- [ ] スキルチェックリストの基礎項目がすべて埋まった

## Phase 3：初案件獲得（Day 22〜30）

### 目標
初めての案件を受注し、納品まで完了する。

### Day 22〜24：プロフィール完成 & 応募準備

| アクション | 詳細 |
|-----------|------|
| プロフィール完成 | テンプレート集を使って自己紹介文を完成 |
| 本人確認 | クラウドワークスの本人確認・NDA同意を完了 |
| 提案文を用意 | ベースの提案文テンプレートをカスタマイズ |

### Day 25〜27：集中応募期間

| アクション | 詳細 |
|-----------|------|
| 案件を探す | 「新着順」で自分のスキルに合う案件を毎日チェック |
| 1日3件以上応募 | 返信率は10〜20%なので数が重要 |
| 差別化する | 相手のチャンネル/サイトを分析して提案文に反映 |
| テスト提案 | 「まず1本テストで」を積極的に提案 |

### Day 28〜30：初案件の受注 & 納品

| アクション | 詳細 |
|-----------|------|
| 条件確認 | 納期・修正回数・納品形式を事前に合意 |
| 50%報告 | 作業の中間段階でクライアントに進捗共有 |
| 納品 | 丁寧なメッセージと共に納品物を提出 |
| 評価獲得 | ★5 の評価をもらうことを意識 |

> **ポイント**: 初案件は「稼ぐ」よりも「実績を作る」ことが目的。相場より少し安くても、丁寧に取り組んで★5評価を取る。

### Phase 3 チェックリスト
- [ ] プロフィールが100%完成した
- [ ] 10件以上の案件に応募した
- [ ] 1件以上の案件を受注した
- [ ] 初案件を納品した
- [ ] クライアントから★4以上の評価を獲得した

## Phase 4：安定受注・スキルアップ（Day 31〜60）

### 目標
月3〜5万円を安定的に稼ぐ。

### Day 31〜40：実績を積む

| アクション | 詳細 |
|-----------|------|
| 継続応募 | 毎日1〜2件は新規案件に応募を続ける |
| リピート打診 | 納品後に「今後もお仕事があればぜひ」と一言添える |
| 実績3件突破 | ★5評価を3件以上集める → 信頼度が大幅UP |

### Day 41〜50：単価を上げる

| アクション | 詳細 |
|-----------|------|
| 相場通りの価格に戻す | 初期の割引価格から通常価格に戻す |
| 提案文を磨く | 実績を提案文に入れる「類似案件の経験あり」 |
| プロフィール更新 | 実績数・評価・ポートフォリオを最新版に |
| 高単価案件に挑戦 | ランサーズやココナラにも展開 |

### Day 51〜60：スキルの幅を広げる

| アクション | 詳細 |
|-----------|------|
| 応用スキル学習 | カリキュラムの中級〜上級編に進む |
| スキルの掛け合わせ | 2つ目のスキルの基礎を学び始める |
| 単価UP交渉 | 継続クライアントに「次回から○○円で」と相談 |

**スキルの掛け合わせ例:**

| メイン | サブ | できるようになること |
|-------|------|------------------|
| 動画編集 | デザイン | サムネ込みのワンストップ提案 |
| ライティング | SNS運用 | 記事 + SNS投稿のセット提案 |
| プログラミング | デザイン | デザインからコーディングまで一括受注 |
| デザイン | ライティング | LPのコピー + デザインをセット提案 |

### Phase 4 チェックリスト
- [ ] 実績が3件以上になった
- [ ] 単価を初期から20%以上UPした
- [ ] リピートクライアントが1名以上いる
- [ ] 月3万円以上の副業収入を達成した
- [ ] 2つ目のスキルの学習を開始した

## Phase 5：月10万円達成・仕組み化（Day 61〜90）

### 目標
月10万円の副業収入を達成し、継続的に稼げる仕組みを作る。

### Day 61〜70：収入源を複数化

| アクション | 詳細 |
|-----------|------|
| 複数プラットフォーム | クラウドワークス + ランサーズ + ココナラ の3つで案件を回す |
| 直接取引 | 信頼関係のあるクライアントとプラットフォーム外で直接契約（手数料削減） |
| パッケージ化 | 「動画編集 + サムネ5枚セット ○○円」のようなセット商品を作る |

### Day 71〜80：効率化・仕組み化

| アクション | 詳細 |
|-----------|------|
| テンプレ化 | よく使うテロップスタイル / バナーテンプレ / 記事構成を型にする |
| 作業時間の短縮 | 同じ品質を半分の時間で出せるようにする → 時給UP |
| 月額契約 | 「月○本の動画編集で月額○万円」の継続契約を目指す |

### Day 81〜90：月10万円の壁を超える

| アクション | 詳細 |
|-----------|------|
| 単価交渉 | 全クライアントに単価見直しを相談 |
| 高単価案件のみに絞る | 低単価案件を断り、時給の高い案件に集中 |
| 発信を始める | X（Twitter）で学びや実績を発信 → 直接依頼の導線を作る |

**月10万円の達成パターン例:**

| パターン | 内訳 | 合計 |
|---------|------|------|
| 動画編集 | 8,000円/本 × 13本 | 104,000円 |
| デザイン | バナー5,000円 × 10枚 + LP 50,000円 × 1件 | 100,000円 |
| プログラミング | LP 50,000円 × 2件 | 100,000円 |
| ライティング | 文字単価2円 × 5,000字 × 10本 | 100,000円 |

### Phase 5 チェックリスト
- [ ] 月10万円の副業収入を達成した
- [ ] 直接取引のクライアントが1名以上いる
- [ ] 月額契約（継続案件）を1つ以上持っている
- [ ] 作業の効率化が進み、初期より半分以下の時間で同品質を出せる
- [ ] X（Twitter）で副業の学びを発信し始めた
