# 案件獲得ロードマップ

> ゼロから初案件を獲得するまでの完全ステップガイド。
> 第1話〜第4話を最後まで視聴してくださった方への限定プレゼントです。
>
> Skill Hacks 全話視聴者限定特典

---

## このロードマップの使い方

1. **STEP 1〜7を順番に進めてください**（飛ばさない）
2. 各STEPの「完了条件」を満たしたら次に進む
3. 迷ったら伴走支援メンバーに相談する
4. **目安期間：2週間〜1ヶ月で初案件獲得**

---

## 全体マップ

```
STEP 1（1日目）    → スキルを選ぶ
STEP 2（2〜7日目）  → コアスキルだけ学ぶ
STEP 3（5〜7日目）  → サンプルを作る
STEP 4（7〜8日目）  → プロフィールを整える
STEP 5（8〜10日目） → 案件に応募する
STEP 6（10〜14日目）→ 初案件を納品する
STEP 7（14日目〜）  → リピートと単価アップ
```

---

## STEP 1：スキルを選ぶ（1日目）

### やること
自分が取り組むスキルを1つ決める。

### 選び方のポイント

| 判断基準 | 質問 |
|---------|------|
| 興味 | どの作業が「やってみたい」と思えるか？ |
| 適性 | 細かい作業が好き？ 文章を書くのが好き？ |
| 市場 | クラウドワークスでどのくらい案件が出ているか？ |
| 速さ | 最短で案件が取れるのはどれか？ |

### 各スキルの特徴

| スキル | こんな人向け | 初案件までの目安 | 初期の案件単価 |
|-------|------------|----------------|-------------|
| 動画編集 | コツコツ作業が好き | 1〜2週間 | 3,000〜5,000円/本 |
| Webデザイン | ビジュアルにこだわりたい | 2〜3週間 | 3,000〜5,000円/枚 |
| プログラミング | 論理的思考が得意 | 2〜4週間 | 3〜5万円/ページ |
| ライティング | 文章を書くのが好き | 1〜2週間 | 文字単価1円〜 |

> **迷ったら？** 伴走支援の初回面談で一緒に決めましょう。全講座学び放題なので、合わなかったら後から変更もOKです。

### 完了条件
- [ ] 取り組むスキルを1つ決めた

---

## STEP 2：コアスキルだけ学ぶ（2〜7日目）

### やること
選んだスキルのカリキュラムで、**案件獲得に必要な最低限の部分だけ**を学ぶ。

### 学習のルール

1. **全部やろうとしない** — カリキュラムの「基礎編」だけでOK
2. **手を動かしながら学ぶ** — 見るだけでなく、必ず自分で操作する
3. **分からないところは即質問** — Q&Aサポートを活用する
4. **1日の学習時間を決める** — 最低1時間、できれば2〜3時間

### 各スキルの「最低限ここまで」ライン

**動画編集の場合:**
- テロップ挿入 → カット編集 → BGM挿入 → SE挿入 → 画像挿入
- この5つができればYouTube編集案件は受けられる

**Webデザインの場合:**
- Photoshopの基本操作 → バナー制作 → SNS画像制作
- バナーが作れれば案件に応募できる

**プログラミングの場合:**
- HTML/CSS → レスポンシブ対応 → 簡単なJavaScript
- LP1ページをコーディングできれば案件に応募できる

**ライティングの場合:**
- SEOの基礎 → 記事構成の作り方 → PREP法での執筆
- 3,000字の記事を書ければ案件に応募できる

### 完了条件
- [ ] カリキュラムの基礎編を一通り完了した
- [ ] 基本的な操作を自分一人でできるようになった

---

## STEP 3：サンプルを作る（5〜7日目）

### やること
案件応募用のサンプル作品（ポートフォリオ）を作る。

### サンプルの作り方

| スキル | 作るもの | 参考にするもの |
|-------|---------|-------------|
| 動画編集 | YouTube動画の編集サンプル1本 | 人気YouTuberの動画をお手本に |
| デザイン | バナー3枚 + サムネイル2枚 | 実際の広告やYouTubeサムネを参考に |
| プログラミング | 架空のLP1ページ | 実在するLPを模写 |
| ライティング | SEO記事2本（各3,000字） | 検索上位の記事を分析して書く |

### サンプル制作のコツ

1. **実在する案件を想定して作る** — 架空クライアントの依頼だと思って取り組む
2. **完璧じゃなくていい** — 70%の完成度でOK。案件を回しながら上達する
3. **見せ方を工夫する** — GoogleドライブやNotionにまとめてURLで共有できるようにする

### 完了条件
- [ ] サンプル作品を規定数作成した
- [ ] ポートフォリオURLとしてまとめた（Googleドライブ / Notion / サイト等）

---

## STEP 4：プロフィールを整える（7〜8日目）

### やること
クラウドソーシングサイトに登録し、プロフィールを完成させる。

### 登録先
1. **クラウドワークス**（案件数が最も多い）
2. **ランサーズ**（高単価案件が多い傾向）
3. **ココナラ**（自分からサービスを出品できる）

> 最初はクラウドワークスだけでOK。慣れてきたら他も登録。

### プロフィール完成チェックリスト
- [ ] プロフィール写真を設定した（顔写真 or プロっぽいアイコン）
- [ ] 自己紹介文を記入した（テンプレート集を活用）
- [ ] スキル・経歴を記入した
- [ ] ポートフォリオURLを掲載した
- [ ] 本人確認（身分証提出）を完了した
- [ ] NDA（秘密保持契約）に同意した

### 完了条件
- [ ] クラウドワークスのプロフィールが100%完成している
- [ ] ポートフォリオURLがプロフィールに掲載されている

---

## STEP 5：案件に応募する（8〜10日目）

### やること
実際に案件を探して応募する。**最低10件に応募**する。

### 案件の探し方

1. クラウドワークスで自分のスキルに合うカテゴリを開く
2. 「新着順」でソートする
3. 以下の条件で絞り込む：
   - 予算が明記されている
   - 「初心者歓迎」「未経験OK」と書いてある
   - 依頼内容が具体的（曖昧な案件は避ける）
   - クライアントの評価が★4以上

### 応募のルール

| ルール | 理由 |
|-------|------|
| 1日3件以上応募する | 返信率は10〜20%。数を打つのが大事 |
| テンプレートをベースに毎回カスタマイズ | コピペ感が出ると落ちる |
| 相手の案件内容に必ず触れる | 「ちゃんと読んでいる」と伝わる |
| テスト納品を提案する | クライアントのリスクを下げる |
| 最初の3件は安くてもOK | 実績（★5レビュー）を作ることが最優先 |

### 応募後にやること

- **返信が来たら即レス**（24時間以内、できれば数時間以内）
- **質問されたら丁寧に具体的に答える**
- **条件が折り合えば、すぐに着手する旨を伝える**

### 完了条件
- [ ] 10件以上の案件に応募した
- [ ] 1件以上の案件を受注した

---

## STEP 6：初案件を納品する（10〜14日目）

### やること
受注した案件を納品する。**ここが一番大事なSTEP。**

### 案件進行の流れ

```
1. 受注確定 → 条件・納期を確認
2. 不明点を質問 → 作業開始前に疑問をゼロにする
3. 作業開始 → 途中経過を共有（50%時点で一度確認）
4. 初稿提出 → 修正依頼を受ける
5. 修正対応 → 最終確認
6. 納品完了 → 評価をもらう
```

### 初案件で意識すべき3つのこと

1. **レスポンスの速さ** — メッセージは即返信。これだけで信頼度が段違い
2. **納期厳守** — 余裕を持ったスケジュールを提示し、前倒しで納品する
3. **丁寧なコミュニケーション** — 進捗報告をこまめにする

### トラブル対応

| トラブル | 対応 |
|---------|------|
| 作業が予想より難しい | 伴走支援メンバーに即相談。一人で抱え込まない |
| 納期に間に合わなそう | 分かった時点ですぐクライアントに連絡。代替案を提示 |
| 修正が多すぎる | 「修正は○回まで」を事前に合意しておく |
| クライアントの返信が遅い | 2営業日返信がなければ丁寧にリマインド |

### 完了条件
- [ ] 初案件を無事に納品した
- [ ] クライアントから★4以上の評価をもらった

---

## STEP 7：リピートと単価アップ（14日目〜）

### やること
初案件の実績を武器に、継続案件と単価アップを目指す。

### 単価アップのロードマップ

```
実績0件     → 相場の70〜80%で受注（★5レビューを集める）
実績3件     → 相場通りの価格に戻す
実績10件    → 相場の110〜120%に値上げ
実績30件以上 → 相場の150%以上、または固定クライアント契約
```

### やるべきこと

1. **納品後にリピートを提案する**（テンプレート集の「リピートのお願い」を活用）
2. **プロフィールに実績を追加する**（「○件の実績・平均★4.8」等）
3. **提案文に実績を入れる**（「類似案件の経験があります」）
4. **スキルの掛け合わせを始める**（動画編集 × サムネデザイン → ワンストップ提案）
5. **直接取引（継続契約）を目指す**（クラウドソーシングの手数料をカットできる）

### スキルの掛け合わせ例

| 掛け合わせ | できる案件 | 単価の目安 |
|-----------|-----------|-----------|
| 動画編集 × デザイン | YouTube運営代行（編集+サムネ） | 1万〜3万円/本 |
| ライティング × デザイン | LP制作（コピー+デザイン） | 10〜30万円/件 |
| プログラミング × デザイン | Webサイト制作（デザイン+コーディング） | 10〜50万円/件 |
| 動画編集 × マーケティング | YouTubeチャンネル運営 | 月額5〜20万円 |
| ライティング × SNS運用 | SNS運用代行 | 月額3〜10万円 |

### 完了条件
- [ ] 継続案件またはリピート案件を獲得した
- [ ] 初案件より高い単価で受注できた
