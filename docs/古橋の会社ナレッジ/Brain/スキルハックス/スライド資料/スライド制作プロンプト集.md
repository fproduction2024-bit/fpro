# スキルハックス スライド制作プロンプト集

> 各スライドをAI（Canva AI / Gamma / ChatGPT等）で制作する際のプロンプトです。
> **共通デザイン指示**を冒頭に記載し、各スライドの個別プロンプトと組み合わせて使用してください。

---

## 共通デザイン指示（全スライド共通で冒頭に付与）

```
【デザインルール】
- スタイル: モダン・ミニマル、プレミアム感のあるビジネスセミナー用スライド
- 配色: ダークネイビー(#1a1a2e)ベース、アクセントにゴールド(#d4a853)とホワイト
- フォント: 見出しはゴシック体（太字）、本文は読みやすいゴシック体
- レイアウト: 余白を十分に取り、1スライド1メッセージを徹底
- アスペクト比: 16:9
- ブランド要素: 右下に「Skill Hacks」ロゴを小さく配置
- 話者: 迫 佑樹（さこ ゆうき）
```

## 第一話（17枚）

### 1-01｜タイトル画面
```
セミナー動画のタイトルスライドを作成。
中央に大きく「なぜ『勉強熱心な人』ほど副業で1円も稼げないのか」と表示。
下部に「Skill Hacks 特別動画講座 第1話」「話者：迫 佑樹」と小さく表示。
背景はダークグラデーション、中央にスポットライト風の光演出。
```

### 1-02｜3つの価値
```
箇条書きスライド。タイトル「この動画シリーズで手に入る3つのこと」。
①スキルを最短最速で身につける方法
②売上を伸ばすための知識
③自立した事業者になるロードマップ
各項目の左にゴールドの番号アイコン。背景ダーク。
```

### 1-03｜半年後のあなた
```
ビジュアル重視のスライド。タイトル「半年後のあなた」。
ノートPC・カフェ・自由な働き方をイメージする写真を背景にぼかし配置。
中央にキャッチコピー「自分の力で稼いだ、という確信」をゴールド文字で大きく表示。
```

### 1-04｜最悪の未来
```
コントラストを意識したスライド。タイトル「逆に、最悪の未来は？」。
背景をやや暗いグレーに変更し、警告感を演出。
中央に「ノウハウコレクターのまま、ただ時間だけが過ぎていく」と白文字で表示。
下部に灰色で「SNSをスクロールしているだけで1年が経つ」。
```

### 1-05｜自己紹介（実績一覧）
```
自己紹介スライド。左側に話者の写真配置エリア。
右側に実績を箇条書き：
・Brain運営（利用者32万人・年間流通10億円超）
・株式会社スキルハックス代表（経営8年目）
・累計受講者3万人以上
・令和の虎 スキル獲得版 主催
・著書『人生攻略ロードマップ』ベストセラー
各項目の左にチェックマークアイコン。
```

### 1-06｜最初からうまくいったわけではない
```
転換スライド。背景を少し暗く。
中央に大きく「でも、最初からうまくいったわけではありません。」
下部に小さく「──ブログ1年間で稼げたのは、たったの273円」。
テキストにフェードイン演出の指示。
```

### 1-07｜ターニングポイント
```
ストーリーテリング用スライド。タイトル「ターニングポイント」。
タイムライン形式で左から右へ：
273円（1年間）→ プログラミングスクール入学 → 初案件獲得 → ○○ハックス創業 → Brain創業 → 年商10億
矢印で繋ぎ、ターニングポイントを星マークで強調。
```

### 1-08｜でも本当にできるの？
```
視聴者の不安を代弁するスライド。
吹き出し風デザインで3つの声を配置：
「本当にそんなこと、自分にできるの？」
「一部のすごい人だけの話でしょ？」
「情報商材って結局詐欺じゃないの？」
背景はダーク、吹き出しは半透明ホワイト。
```

### 1-09｜なぜスキル提供なのか？
```
タイトルスライド。大きく「なぜスキル提供なのか？」。
下部にサブテキスト「パソコン1台・在庫なし・初期投資ゼロ」を3つのアイコン付きで横並び。
アイコン：💻 パソコン / 📦 在庫不要 / ¥0 初期費用ゼロ
```

### 1-10｜成果が出る人 vs 出ない人
```
2カラム比較スライド。
左カラム（赤系・暗め）「成果が出ない人」：枝葉の情報を追いかける、断片的に情報収集、どれも中途半端
右カラム（ゴールド系・明るめ）「成果が出る人」：最低限のコアスキルに集中、体系化された幹を学ぶ、すぐ実践
中央に VS を大きく配置。
```

### 1-11｜必須5項目
```
アイコン付きリストスライド。タイトル「案件獲得に必要なのは、たった5つ」。
5つを横並びでアイコン付き表示：
🔤 テロップ / ✂️ カット / 🎵 BGM / 🔊 音入れ / 🖼️ 画像挿入
下部に「この5つができれば、YouTube編集案件は受注できます」。
```

### 1-12｜プレゼント予告
```
特典紹介スライド。タイトル「特別プレゼント予告」。
チェックリストのイメージ画像を中央に配置。
「スキルチェックリスト」をゴールド文字で大きく。
下部に「動画シリーズ最終話の後にお届けします」。ギフトボックスのアイコン。
```

### 1-13｜足りないもの
```
問いかけスライド。中央に大きく
「もし1つだけ、あなたに足りないものがあるとしたら──」
下部に答え：「"最低限の情報"を網羅しているコンテンツ」をゴールドで強調。
```

### 1-14｜スキル3：その他7
```
円グラフまたはドーナツチャート。タイトル「副業で稼ぐ能力の比率」。
スキル：30%（ブルー系）/ その他（ソフトスキル・人間力）：70%（ゴールド系）
右側に「その他」の具体例を箇条書き：案件応募、クライアント対応、納品段取り、コミュニケーション
「スキルだけでは1円にもならない」を赤テキストで強調。
```

### 1-15｜理想の未来
```
ビジュアルスライド。タイトル「理想の未来」。
自由に働く人のイメージ（カフェでPC作業、旅先での仕事など）を背景にぼかし配置。
中央に「スキル習得 + 案件獲得 + 納品 = すべてをカバー」のフレーズ。
```

### 1-16｜次回予告
```
予告スライド。タイトル「次回 第二話」。
サブタイトル「スキルハックスが提供する内容を公開」。
2つのポイントを表示：
①スキルの部分の具体的な内容
②スキル以外をどのようにカバーしているか
背景にうっすら「2」の大きな数字を透かし。
```

### 1-17｜CTA
```
行動喚起スライド。背景にゴールドのアクセントを多めに。
中央に大きく「感想を送って、第二話を受け取る」。
下部に矢印とボタン風デザイン「▼ 今すぐ感想を送る ▼」。
注意書き「この動画シリーズは3日間限定配信です」を赤テキストで表示。
```

```
「その他」部分にスポットライト演出で強調。
```

### 2-06｜「その他7」の内訳リスト
```
リストスライド。タイトル「スキル"以外"に必要なこと」。

📝 案件への応募方法 / 💬 クライアントとのコミュニケーション / 📋 納品の段取り
💰 価格設定・見積り / 📅 スケジュール管理 / 🤝 信頼関係の構築 / 📊 自己管理

```

### 2-07｜一人で手探り→不安→挫折
```
フロー図スライド。タイトル「なぜ90%が挫折するのか」。
左から右への3ステップフロー：
「一人で手探り」→「不安が膨らむ」→「挫折」
各ステップを丸いノードで繋ぎ、矢印で接続。最後の「挫折」は赤く強調。
下部に対比：「仲間＋サポート → 安心 → 継続 → 成果」を緑系で表示。
```

### 2-08｜3つの武器 タイトル
```

```

### 2-09｜学びのピラミッド図
```
ピラミッド図スライド。タイトル「武器① コアスキルだけに集中する」。
ピラミッド構造：上段「枝葉のテクニック（不要）」/ 中段「応用スキル」/ 下段（基盤）「コアスキル（ここだけで十分）」。
下段をゴールドで強調、上段はグレーアウト。
右側にテキスト「まず"幹"を固める。枝葉は後からいくらでも伸びる」。
```

### 2-10｜常識と逆の順番
```
2カラム比較スライド。タイトル「武器② 技術が習熟する前にお金をもらう」。
左カラム（×）「一般的な順番」：勉強する → 完璧にする → 案件を取る（永遠に実践しない）
右カラム（◎）「正しい順番」：最低限学ぶ → 案件を取る → 実践しながら上達する
右カラムをゴールドの枠で囲んで強調。
```

### 2-11｜正のスパイラル図
```
循環図スライド。タイトル「稼ぎながらスキルが磨かれる正のスパイラル」。
円形の矢印で循環を表現：
「案件を受ける」→「実践する」→「フィードバックを得る」→「スキルが上がる」→「単価が上がる」→（循環）
中央に上向き矢印で「収入もスキルも右肩上がり」。
```

### 2-12｜迫の初案件ストーリー
```
ストーリーテリングスライド。タイトル「僕の初案件の話」。
吹き出しまたはタイムライン形式で：
「そもそも1案件目がうまくいくはずない」→「でもやった」→「実践から学んだことが次に活きた」
```

### 2-14｜こんな人から学ぶと失敗する

```
警告風スライド。タイトル「こんな人から学ぶと失敗する」。
```

### 3-01｜タイトル画面
```
第三話のタイトルスライド。中央に動画タイトルを大きく表示。
下部に「Skill Hacks 特別動画講座 第3話」「話者：迫 佑樹」。
```

### 3-02｜でも本当に自分にもできる？
```
共感スライド。中央に大きく「でも本当に、自分にもできる？」。

「特別な才能がないとダメなんでしょ？」「独学で挫折した経験がある…」
```

### 3-03｜2軸マトリクス（空）
```
図解スライド。タイトル「市場の構造を見てみましょう」。
縦軸「サポートの手厚さ」（下:薄い → 上:手厚い）
横軸「価格」（左:安い → 右:高い）
2本の軸だけを表示した空のマトリクス。次のスライドで中身を埋める構成。
```

### 3-04｜マトリクスに4象限を配置
```
3-03のマトリクスに4象限の内容を配置：
右上「高額スクール（80万〜）手厚いサポートだが高い」
左上「高額コンサル（100万〜）少人数で手厚い」
右下「（空白）」← ここが狙い目
左下「安い教材（数万円）サポートなし・独学」
各象限にラベルと簡単な説明文。
```

### 3-05｜ここがスキルハックス
