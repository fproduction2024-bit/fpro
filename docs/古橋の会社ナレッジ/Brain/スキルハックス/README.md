├── プロット/                         ← 各話のプロット（構成案）

│   ├── 第三話_台本.md

├── スライド資料/                     ← スライド制作関連

    ├── ミーティングメモ.txt
    ├── スキルハックスファネルミーティングメモ.txt
    ├── スキルハックスファネルミーティング録音ver.txt
